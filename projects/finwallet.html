FinWallet

Keeping track of finances to raise financial awareness.

ABOUT

A simple personal finance tracker made with React. Here one can add income and expenses and can view all transactions as well as the current balance. Apart from this, there is a finance page that shows details about crypto, top gainers in both NSE and BSE stocks, and the latest news. The web app uses Firebase Authentication for user login, signup, and logout and uses Firebase Realtime Database for storing user data. The app is hosted in Netlify.
